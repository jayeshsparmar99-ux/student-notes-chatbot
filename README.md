# AI Notes Chatbot

An AI-powered study assistant built with Streamlit and Google Gemini.

## Features

- Upload PDF notes
- Ask questions from notes
- Generate MCQs
- Generate important questions
- Generate summaries
- Chat history

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyMuPDF (fitz)
